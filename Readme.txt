<h1>Subscript is a Jenkins UI Wrapper designed for projects at my company. </h1>

It will take all the projects from jenkins:
<ul>
<li>Allow the user to choose what projects, branches and builds they want to display on a UI</li>
<li>The UI will allow you to download builds</li>
<li>Check changelists</li>
<li>Store builds in a well formatted structure</li>
<li>Delete globabl preferences</li>
<li>Unzip utility.</li>
</ul>
